# Development

This game was created using **Vibe Coding** with GitHub Copilot.

Vibe Coding is an AI-assisted development approach where the developer collaborates with AI to rapidly prototype, iterate, and build complete projects through natural conversation and intelligent code generation.

## Technology Stack
- Pure JavaScript (ES6+)
- HTML5 Canvas for rendering
- Web Audio API for sound
- CSS3 for UI styling

## Project Structure
```
Story Balloon/
├── index.html          # Main game file
├── css/
│   ├── style.css       # Game UI styles
│   └── preloader.css   # Loading screen styles
├── js/
│   ├── game-new.js     # Main game engine
│   ├── balloon.js      # Player balloon logic
│   ├── obstacles.js    # Obstacle classes
│   ├── items.js        # Collectible items
│   ├── stage-objects.js # Stage-specific objects
│   ├── sound-manager.js # Audio management
│   └── preloader.js    # Asset loading
├── assets/             # Logos and images
└── music/              # Background music files
```
